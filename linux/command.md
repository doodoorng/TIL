# basic command

> 리눅스 기본 명령어를 정리해봅시다.



### 이동

##### cd(change directory)

- 폴더를 이동하는 명령어
- cd <directory name>

##### pwd(print working directory)

- 현재 나의 위치를 출력

### 생성

##### Mkdir(make directory)

- 폴더를 생성하는 명령어
- Mkdir <directory name>



##### touch

- 파일을 생성하는 명령어
- touch. `ls -a`
- `ls- a`
